##  Project Architecture

The project follows a modular React architecture.

```
marwane-portfolio
│
├── public
│
├── src
│
│   ├── assets
│   │   └── marwane.jpg
│
│   ├── components
│   │   ├── Navbar.jsx
│   │   ├── Footer.jsx
│   │   └── GithubActivity.jsx
│
│   ├── pages
│   │   ├── Home.jsx
│   │   ├── About.jsx
│   │   ├── Skills.jsx
│   │   ├── Projects.jsx
│   │   └── Contact.jsx
│
│   ├── data
│   │   ├── projects.js
│   │   └── skills.js
│
│   ├── styles
│   │   └── global.css
│
│   ├── App.jsx
│   └── main.jsx
│
└── package.json
```

This structure improves **code readability and scalability**.
